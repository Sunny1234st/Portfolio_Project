Project 1

Music Player Application Developed a desktop music player using Python's Tkinter for the graphical user interface and Pygame for audio playback. This application allows users to: Load Audio Files: Select a directory containing MP3 files to populate the playlist. Play, Pause, and Navigate: Control music playback with play, pause, next, and previous buttons. User Interface: The application features a simple and intuitive interface with buttons for controlling playback and a listbox to display available songs. Key technologies: Python, Tkinter, Pygame
